Wired-Tool: User tragen


​Wired-Tool

WIRED Extra: User tragen

​​Daten
Wired-Name: User tragen
Wiredtyp: Extra
Auslösequelle: Benötigt KEINEN auslösenden Typ (Habbo, Bot, Haustier).
​
Funktion: Alle Möbel, die durch Wireds bewegt werden, können User transportieren, wenn das Add-on sich in demselben Stapel befindet.

Definition: Jedes Möbel, dass durch einen Wired-Effekt bewegt oder versetzt wird, kann Habbos, Bots und Haustiere transportieren. Das schließt alle Effekte dieser Art ein. Das Transportieren findet auch dann statt, wenn mehrere sich auf denselben Möbel befinden. Unter "User werden getragen wenn:..." gibt es 2 Optionen. 1. "...sie direkt auf dem sich bewegenden Möbel stehen". Diese Option erlaubt es User nur zu tragen, wenn sie direkt auf den bewegenden Möbel stehen und sich keine anderen Möbelstücke zwischen dem User und dem bewegenden Möbelstück befinden​. 2. "...sie auf der gleichen Fliese wie das sich bewegende Möbel stehen". Diese Option erlaubt das Tragen immer, auch wenn Möbelstücke sich über den User befinden. Wenn ein User über einen Möbel fährt, welches auf Habbos reagiert, dann wird die Reaktion ausgeführt ohne Tritt-auf auszulösen. Z. B. fährt man über einen Fußball, dann wird dieser geschossen, aber Tritt-auf Wired wird nicht ausgelöst oder Banzai Boden wird gefärbt ohne Tritt-auf.
​
Erweiterte Einstellungen: "Tragen erlauben für" kann weiter eingestellt werden.

Verwandte Wireds: -
Anwendungsbereich: Verschiedene Transportmöglichkeiten wie z. B. Fahren oder Schweben.

​
Videobeschreibung: Unterschiede zwischen Option 1 und 2.
