Wired-Tool: Zähler kontrollieren


​Wired-Tool

WIRED Effekt: Zähler kontrollieren

​​Daten
Wired-Name: Zähler kontrollieren
Wiredtyp: (Timer-) Effekt
Auslösequelle: Benötigt KEINEN auslösenden Typ (Habbo, Bot, Haustier).
​
Funktion: Alle (Countdown-)Timer und Zähler können kontrolliert werden: Starten, Stoppen, Zurücksetzen, Pausieren und Fortsetzen (Effektzeit 0 - 10 s).
​
Definition: Wird ein Spiel-Timer gestartet, wird Spiel beginnt ausgelöst. Falls der Timer bereits läuft, wird das Spiel beendet und Spiel zuende wird ausgelöst, sofort danach wird fast gleichzeitig Spiel beginnt ausgelöst und der Timer startet erneut. Wird ein Timer gestoppt, dann wird nur Spiel zuende ausgelöst und der Timer hält an. Wird der Timer zurückgesetzt während er läuft, wird Spiel zuende ausgelöst und die Anzeige wird auf den Anfangszustand gebracht. Wenn der Timer aber zuende gelaufen ist und beendet wurde, dann wird nur die Anzeige auf den Anfangszustand gebracht. Wird ein Timer Pausiert hält der Timer nur an ohne jemals das Spiel zu beenden. Wird der Timer fortgesetzt, dann wird das pausieren aufgehoben, aber es findet kein "Spiel beginnt" statt. Wird der Timer aber dann fortgesetzt, wenn er vorher gestoppt und ein Spiel beendet wurde, dann wird er ebenfalls an der Stelle weiter laufen an dem er gestoppt wurde, jedoch ohne ein Spiel zu starten. Zähler, die nicht an einem Spiel geknüpft sind (Zähl-Timer) werden weder das Spiel beginnen noch beenden, die Zähler-Kontrolle bleibt ansonsten gleich wie bei den Spiel-Timer.

Erweiterte Einstellungen: Möbelquelle kann weiter eingestellt werden.
​
Verwandte Wireds: Zähler erreicht programmierte Zeit
Anwendungsbereich: Timer und Zähler Steuerungen, Timingkontrolle, automatische Reset-Timer in Games

​
Videobeschreibung: Im Video wird gezeigt wie ein Spiel-Timer laufen kann ohne, dass ein Spiel selbst laufen muss. Anfänglich wird das Spiel kurz gestartet aber sofort wieder beendet und dann läuft der Timer weiter.
