WIRED Auslöser: User klickt Möbel


​​Daten
Wired-Name: User klickt Möbel​, kurz: Klick-Wired
Wiredtyp: Auslöser
Auslösequelle: Benötigt einen Habbo als auslösenden Typ.

Funktion: Wired-Stapel wird ausgelöst, wenn ein User auf einen der ausgewählten Möbel klickt.

Definition: Der Auslöser erkennt jeden Einzel-Klick (Single-Click) eines Users. Dafür muss sich der Habbo nicht unbedingt neben dem ausgewählten Möbel befinden, sondern er kann dies aus der Ferne tun.

Erweiterte Einstellungen: Möbelquelle kann weiter eingestellt werden.
​
Verwandte Wireds: Möbel wird benutzt
Anwendungsbereich: Wird oft in Verbindung mit einem Steuerkreuz verwendet.

​
Videobeschreibung: Ein klick auf einen Blob und es ist als hätte man darauf getreten!
