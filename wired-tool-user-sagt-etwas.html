WIRED Auslöser: User sagt etwas

​​Daten
Wired-Name: User sagt etwas, kurz: Sagt-was
Wiredtyp: Auslöser
Auslösequelle: Benötigt einen auslösenden Typ (Habbo, Bot, Haustier). Auslösequelle vom Nachricht-Stapel kann empfangen werden.
​
Funktion : Wired-Stapel wird ausgelöst, wenn der auslösender Typ etwas flüstert, sagt oder schreit, das im Sagt-etwas Wired gespeichert ist.

Definition: Ein Habbo kann auf "natürliche" Weise durch die Chatbox etwas sagen. Das Nachricht anzeigen Wired bringt Habbos zum Sprechen, wodurch das Sagt-etwas ausgelöst wird. Außerdem kann der Habbo noch mit besonderen Möbeln zum Sprechen erzwungen werden (siehe hier) und mit dem Benutzerrauswurf-Wired, sofern man kein Raumadmin ist. Bots können das Sagt-etwas durch den Chatter, die Bot spricht und Bot flüstert Wireds und die automatische Sprachausgabe auslösen (also das was die Dienst- und Registrier-Bots von allein sagen). Haustiere lösen das Sagt-etwas aus, wenn sie automatisch etwas sagen oder trainiert werden, wobei beim Trainieren der Habbo der Auslöser ist. UPDATE INFO: Es wurde eine 2. Option hinzugefügt, wodurch es möglich ist, dass ein Habbo/Bot/Haustier den exakten Wortlaut verwenden muss, um auszulösen. D. h. wird im Wired der Satz: "Ich bin bereit" gespeichert und Option 2 ist ausgewählt, dann werden Sätze wie: "Ich bin bereit oder auch nicht..." ignoriert. Es muss also genau das gesagt werden ohne etwas hinzuzufügen.
 
Verwandte Wireds: Nachricht anzeigen

Anwendungsbereich: Exit-Einstellungen in verschiedenen Games, EDW, Code-Wörter für Spielleiter u. ä.

​
Videobeschreibung: Steht im Sagt-etwas Wired das Gleiche wie im Nachricht-Wired, kann eine Dauerschleife beim Auslösen entstehen. Hierbei muss die Effektzeit hoch genug sein.
