Wired-Tool: Mindestens eine Bedingung ist wahr


​Wired-Tool

WIRED Extra: Mindestens eine Bedingung ist wahr

​​Daten
Wired-Name: Mindestens eine Bedingung ist wahr, kurz: Oder-Wired
Wiredtyp: Extra (Bedingung)
Auslösequelle: Benötigt KEINEN auslösenden Typ (Habbo, Bot, Haustier).

Funktion: Wenn eine oder mehrere Bedingungen erfüllt sind, wird der Stapel ausgelöst, worin sich das Extra Wired befindet. UPDATE INFO: Es können bis zu 20 Bedingungen ausgewählt werden von denen mindestens eine erfüllt sein muss. 

Definition: Jede Bedingung (positiv wie negativ) kann mit dem Extra Wired ausgewählt werden und sogar dieses Extra Wired selbst kann gültig ausgewählt werden, sie müssen nicht im selben Stapel vorhanden sein und können sogar einzeln im Raum liegen. Wenn eine Bedingung erfüllt ist, aber alle anderen im selben Stapel (oder was außerhalb ausgewählt wurde) nicht, kann der Stapel ausgelöst werden. Das Extra Wired ist eine Oder-Funktion. Es kann auch mit den anderen beiden Extra Wireds kombiniert werden, ohne dass sie sich gegenseitig beeinflussen. Das Add-on beinhaltet 7 Optionen, welche hier erklärt werden.
UPDATE INFO: Außerdem können auch mehrere Oder-Wireds kombiniert werden. Dabei muss jedes Oder-Wired erfüllt werden, d. h. in jedem Oder-Wired muss immer mindestens eine Bedingung wahr sein. Beispiel: Bedingungen a, b und c sind im 1. Oder-Wired ausgewählt und Bedingungen d und e im 2. Das bedeutet (a oder b oder c) UND (d oder e) muss gültig sein, damit ausgelöst werden kann. 

Erweiterte Einstellungen: Die Möbelquelle kann weiter eingestellt werden.

Verwandte Wireds: -
Anwendungsbereich: Wired-Minimierungen, Vereinfachung von komplexen Einstellungen, Zulassen von verschiedenen Möglichkeiten in einem Spiel
​
Videobeschreibung: Das Video ist für Fortgeschrittene. Sie zeigt wie ein Wired-System die Anzahl an besetzten Flächen angibt. Wenn es in einem Feld verschiedene Bereiche gibt, kann diese Einstellung erkennen wie viele Bereiche besetzt sind und gibt die Anzahl an Besetzungen wieder unabhängig davon in welche Reihenfolge oder Kombination die Besetzung erfolgt.
