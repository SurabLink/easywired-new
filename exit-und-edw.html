​Exit und EDW

Wireds:
WIRED Auslöser: Ein Habbo sagt etwas
WIRED Effekt: Zu Möbelstück teleportieren
WIRED Bedingung: Auslösender Habbo steht auf Möbelstück (EDW)
Allgemein:
In vielen Räumen findet man das Codewort "Exit", um einen bestimmten Bereich im Raum zu verlassen oder in einem Bereich wieder zurückzukehren. Im Leitergame "Errate das Wort" (EdW) wird diese Wired-Kombi auch benutzt außer, dass eine Bedingung (siehe die Einstellung dazu) noch dazukommt.
Einstellung (Exit):
Für den Auslöser Sagt-etwas muss ein Wort ins Wired reingeschrieben werden. Dieses Wort wird das Code-Wort mit, dem ihr teleportiert werdet. In diesem Fall verwenden wir "Exit". Bei "Auslösenden Habbo wählen:" könnt ihr einstellen, dass der Effekt bei allen funktioniert. Dafür wählt ihr dann "Irgendein Habbo" aus. Wenn ihr den unteren auswählt, funktioniert der Effekt nur bei dem Raumbesitzer, auch wenn euer Name darin steht.
Beim Effekt Habbo-Teleportieren wird nur ein Möbel ausgewählt, zu dem ihr teleportieren möchtet. Die Effektzeit unten könnt Ihr für diese Einstellung auf 0 Sekunden lassen.
Die Wireds müssen nun aufeinandergestapelt werden, damit sie funktionieren, so wie hier:
Exit-Wiredkombi
Einstellung (EDW):
Die obige Wiredkombi wird genauso auch hier benötigt. Am besten ist es, dass der Bereich wohin man teleportiert wird mit einem ToW (Tür ohne Wiederkehr) abgesperrt wird, damit keiner ungültig rein kann, so sieht man genau wer durch das richtige Erraten rein kam. 
ToW
Die Bedingung Auslösender Habbo steht auf Möbelstück wird benötigt, damit außerhalb vom Spielfeld keiner mitraten kann, selbst wenn er richtig rät. Im Wired wählt man die Möbel von der Spielfläche aus. Da man nur 5 Möbel auswählen kann ist es hilfreich 5 große (3 mal 3) Teppiche auszuwählen worauf dann z. B. Stühle/Bänke und ähnliches drauf sein können. So kann man eine größere Spielfläche haben. Befindet sich jemand auf diese im Wired ausgewählten Teppiche oder Fliesen, dann können nur diese das Wort erraten. Diese Bedingung kommt dann einfach auf den Stapel dazu. 

Habbo Beta/Unity:



​Hast du gewusst?
Mehrere sagt etwas Wireds können gleichzeitig ausgelöst werden.  Dafür müssen alle Code-Wörter in einer Sprechblase geschrieben werden.
Sagt etwas Wireds können nacheinander oder gleichzeitig ausgelöst werden

Das ist sehr praktisch, wenn man als Leiter in einem Event mehrere Funktionen zusammen aktivieren möchte aber die Stapel getrennt Funktionieren.
