WIRED Auslöser: User klickt auf Fliese

​​Daten
Wired-Name: User klickt auf Fliese, kurz: Klickfliese-Wired
Wiredtyp: Auslöser
Auslösequelle: Benötigt einen Habbo als auslösenden Typ.

Funktion: Wired-Stapel wird ausgelöst, wenn ein User auf einer der ausgewählten Bodenfliesen klickt. 
​
Definition: Der Auslöser erkennt jeden Einzel-Klick (Single-Click) eines Users, der auf eine Bodenfliese klickt. Dafür muss sich der Habbo nicht unbedingt neben der ausgewählten Bodenfliese befinden, sondern er kann dies aus der Ferne tun. Allerdings muss sich auf der Bodenfliese "Unsichtbare Klickfliese"-Möbel befinden, welche im Wired tatsächlich markiert wird, um die Bodenfliese indirekt auszuwählen. 

Erweiterte Einstellungen: Möbelquelle kann weiter eingestellt werden.

Verwandte Wireds: User klickt Möbel
Anwendungsbereich: Klickauslösung auf reine Bodenflächen möglich
Unsichtbare Klickfliese
Videobeschreibung: "User klickt auf Fliese" und "User klickt Möbel" im vergleich.
