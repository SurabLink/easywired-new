WIRED Effekt: Bot folgt User

​​Daten
Wired-Name: Bot folgt User, kurz: Bot-folgt
Wiredtyp: (Bot-) Effekt
Auslösequelle: Benötigt einen Habbo als auslösenden Typ. Auslösequelle kann an Bot erreicht User Stapel weitergeleitet werden.

Funktion: Bestimmter Bot folgt dem auslösenden User oder beendet das Folgen (Effektzeit 0 bis 10 s).

Definition: Der Botname muss genau ins Wired eingetragen werden. Es darf nur ein Bot mit denselben Namen im Raum geben. Der auslösende Habbo wird vom Bot verfolgt bis "Folgen beenden" ausgelöst wurde oder der Raum user-leer neu geladen wird. Das Folgen beenden kann auch von einem User erfolgreich ausgelöst werden, wenn dieser nicht derjenige ist, der verfolgt wird. Es können nur Auslöser in Kombination verwendet werden, die sich auf Habbos beziehen.

UPDATE INFO: Die Groß- und Kleinschreibung des Botnamens ist nicht mehr relevant.
​
Erweiterte Einstellungen: Botquelle und Userquelle können weiter eingestellt werden. 
​
Verwandte Wireds: Bot erreicht User
Anwendungsbereich: ​Bot-Spiele, Bot-Storys, Bot-Steuerung

​
Videobeschreibung: Bot folgt User auch dann noch, wenn die Wireds nach dem Auslösen aus dem Raum geworfen werden und der Habbo den Raum neu betritt. Erst, wenn der Raum für eine längere Zeit userleer bleibt, dann wird das Folgen beendet.
