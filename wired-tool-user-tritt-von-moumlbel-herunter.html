WIRED Auslöser: User tritt von Möbel herunter

​​Daten
Wired-Name: User tritt von Möbel herunter, kurz: Tritt-runter
Wiredtyp: Auslöser
Auslösequelle: Benötigt einen auslösenden Typ (Habbo, Bot, Haustier).

Funktion: Wired-Stapel wird ausgelöst, wenn der auslösende Typ von einem markierten Möbel heruntertritt.

Definition: Während dem Heruntertreten befindet sich der Habbo in Bewegung und noch auf dem Möbel, wovon er heruntertritt. Das Auslösen passiert  0,5 Sekunden bevor der Habbo auf dem nächsten Möbel ist, falls der Habbo per Mausklick herunter geht.
​Rollt man dagegen mit einem Roller ab, verschiebt sich die Phase des Heruntergehens um 0,5 Sekunden. Das bedeutet beim Abrollen wird das Tritt-herunter Wired ausgelöst, wenn der Habbo bereits auf dem neuen Möbel ist.
​
UPDATE INFO: Verlässt ein Habbo den Raum von einer ausgewählten Fliese aus, wird tritt herunter nun ebenfalls ausgelöst. Das gleiche gilt, wenn ein Bot oder Haustier von der Fliese aufgenommen wird.

Erweiterte Einstellungen: Möbelquelle kann weiter eingestellt werden.
​
Verwandte Wireds: Habbo tritt auf Gegenstand
Anwendungsbereich: Als Grundlagen Wired wird es allgemein oft gebraucht, jedoch nicht so oft wie das Tritt-auf Wired. Es wird eher bei besonderen Einstellungen benötigt. Das untere YouTube-Video zeigt eine Beispiel-Einstellung, in der es darum geht Banzai AFK zu leveln.

​
Videobeschreibung: Die Einstellung ist für sich allein nicht vollständig Anwendbar und nur eine Technik, die in AFK-Arenen eingebaut werden kann. Diese Technik ist nicht besonders effizient. Es gibt wesentlich bessere Möglichkeiten, aber mit der Einstellung soll gezeigt werden, was mit dem Tritt-herunter Wired eingestellt werden könnte.
